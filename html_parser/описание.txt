В файле config.ini содержутся данные дял подключения к базе данных. 
В файле database.py код для работы с базой данных.
В файле pars.py код для парсинга данных с сайта.
Файле main.py связывает парсинг и запись в базу данных, запускать нужно только его. 
Файл requirements.txt содержит зависимости для работы скрипта.
